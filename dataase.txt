ans:- MySQL is a widely used relational database management system (RDBMS).

MySQL is free and open-source.

MySQL is ideal for both small and large applications.

MySQL is a very popular open-source relational database management system (RDBMS).



2.What is RDBMS?

ans:- RDBMS stands for Relational Database Management System.

RDBMS is a program used to maintain a relational database.

RDBMS is the basis for all modern database systems such as MySQL, Microsoft SQL Server, Oracle, and Microsoft Access.

RDBMS uses SQL queries to access the data in the database.


3. What is a Database Table?

ans:- A table is a collection of related data entries, and it consists of columns and rows.

A column holds specific information about every record in the table.

A record (or row) is each individual entry that exists in a table.



4. What is a Relational Database?

ans:- A relational database defines database relationships in the form of tables. The tables are related to each other - based on data common to each.


5. what is database ?

ans:- A database is a software that stores data in organized manner.
